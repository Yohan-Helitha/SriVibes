# Fonts Directory

Please download and place the following Montserrat font files in this directory:

1. **Montserrat-Bold.ttf**
2. **Montserrat-Regular.ttf**
3. **Montserrat-SemiBold.ttf**

## Download Source

You can download Montserrat fonts from:
- [Google Fonts](https://fonts.google.com/specimen/Montserrat)
- Direct download: https://fonts.google.com/download?family=Montserrat

## Installation

1. Download the Montserrat font family
2. Extract the zip file
3. Copy the following files to this directory:
   - `Montserrat-Bold.ttf` (from the static folder)
   - `Montserrat-Regular.ttf` (from the static folder)
   - `Montserrat-SemiBold.ttf` (from the static folder)

The app will load these fonts on startup.
